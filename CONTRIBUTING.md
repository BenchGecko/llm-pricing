# Contributing

## Reporting Errors

If you notice incorrect pricing, please open an issue with:
- The model name and ID
- The correct price with source link
- The incorrect price shown in our data

## Data Sources

This data is automatically fetched from [OpenRouter](https://openrouter.ai). If OpenRouter has incorrect data, please report it there first.

## Adding New Fields

Want to see additional data (e.g., rate limits, latency)? Open an issue describing the use case and we'll evaluate adding it.
